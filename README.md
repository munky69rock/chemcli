chemcli
=======

cli tools for chemoinfomatics with less dependencies


Install
-------

```sh
$ pip install git+https://github.com/munky69rock/chemcli.git
```


Usage
-----

```sh
$ sdf sum [SDF file]
$ sdf head [SDF file] -l 1
$ sdf grep [PATTERN] [SDF file]
$ sdf grep -v [PATTERN] [SDF file]
```
